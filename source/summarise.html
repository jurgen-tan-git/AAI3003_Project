summarise module¶

summarise.main()¶

summarise.summarize(text: str, language: str = 'english', sentences_count: int = 5) → str¶

Return a summary of the given text using LSA summarization technique.

Parameters:

  • text (str) – Text to summarize.

  • language (str, optional) – Language for the tokenizer, defaults to “english”

  • sentences_count (int, optional) – Number of sentences to output, defaults to 5

Returns:

Summarized text

Return type:

str
